# Heatmap

The Heatmap visually represents the intensity of workload or 'heat' at each process step and transition in real-time.
By color-coding activity levels, from 'cool blue' for minimal activity to 'fiery red' for high workload, users gain immediate
clarity on where bottlenecks occur, which steps are underutilized, and where process optimizations can be made. With
this tool, businesses can effortlessly identify inefficiencies and streamline operations for enhanced
productivity and performance.

By default, the <TypeLink type="ProcessMining" /> component provides a heatmap which can be disabled.
